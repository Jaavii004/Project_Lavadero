Una tabla en HTML se define utilizando la etiqueta <table>. Dentro de esta etiqueta, definimos filas de la tabla utilizando la etiqueta <tr> (table row).

Cada fila se divide en celdas utilizando la etiqueta <td> (table data/cell) para datos regulares y <th> (table header) para celdas de encabezado.

Los encabezados de la tabla se suelen resaltar visualmente y se definen en la primera fila utilizando <th>.

Los estilos para la tabla y las celdas se pueden aplicar usando CSS.
